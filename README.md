# Java Daily Coding Practice

## Purpose

This repository is my personal Java practice journal. It is dedicated to building strong programming fundamentals before starting Data Structures and Algorithms (DSA).

The goal is to solve basic Java problems every day, strengthen problem-solving skills, and develop confidence with core programming concepts.

---

## Objective

* Build a strong Java foundation from scratch.
* Practice writing clean and readable code.
* Improve logical thinking and problem-solving skills.
* Become comfortable with Java syntax and standard libraries.
* Be fully prepared to begin DSA with confidence.

---

## Practice Plan

I am solving **180+ beginner-friendly Java problems** covering fundamental programming concepts. These problems were generated with the help of an LLM to provide a structured learning path from absolute basics to DSA readiness.

Rather than memorizing solutions, the focus is on understanding the logic behind each problem and solving it independently.

---

## Topics Covered

* Variables & Data Types
* Input & Output
* Operators
* Type Casting
* Conditional Statements (`if`, `if-else`, `switch`)
* Loops (`for`, `while`, `do-while`)
* Pattern Printing
* Methods (Functions)
* Arrays
* Strings
* Searching
* Basic Sorting
* Recursion Basics
* Basic Mathematics
* Number Problems
* Character Problems
* Logical Thinking
* Beginner Problem Solving

---

## Rules I Follow

* Solve every problem without copying solutions.
* Understand *why* the solution works.
* Write clean and readable code.
* Improve the solution whenever possible.
* Practice consistently every day.
* Focus on logic first, optimization later.

---

## Goal

By completing these 180+ practice problems, I aim to:

* Strengthen my Java fundamentals.
* Develop consistent coding habits.
* Improve logical reasoning.
* Build confidence in solving programming problems.
* Be fully prepared to start Data Structures and Algorithms (DSA).
